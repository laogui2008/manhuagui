# 非专业Android开发人员
### 一边看官方文档，一边查资料，手动撸代码。。。。。。

一款简单的观看[漫画柜](https://www.manhuagui.com)漫画APP，没有性能测试，没有兼容性测试。  
  
站点不能访问，需要自备代理。

手头上只有两台机子：android8、android11。在android11上功能还算正常，在android8上有点问题，基于能用就行的原则，不太影响使用的问题，懒得修了。

#### 在android8上发现的问题：
~~1. 看图时，进入全屏，隐藏状态栏。切换后台，再切回来，状态栏会显示。~~  
2. 导出观看记录时，不能直接选择下载目录保存，应用会崩掉退出。需要选择一个子目录才能正常保存。
   
   
### 不能保证在其它机子也能正常使用!!!  
   
   
### 在长滚动列表，双击TAG可以快速回到顶部
### 在观看漫画时，双击中间区域，可以放大图片，再次双击还原。在放大图片时，可以单指拖动图片、双指缩小(放大)图片
   
   
### 更新记录
##### 20260423
1. 在观看漫画时，在第1张、最后1张图片，点击左右两侧区域，根据阅读方向，可以返回阅读上一章节内容(非第1章节)、继续阅读下一章节内容(非最后1章节)。
2. 删除没有用到的依赖包，减少apk大小。
3. 解决“在沉浸式全屏观看漫画时，从前台切换到后台，再切换回到前台，沉浸式不生效”的问题。
4. 在观看漫画时，点击顶部的数字区域，显示跳页输入框，输入数字后，点击按钮，进行页面跳转。
